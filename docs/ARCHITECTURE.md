# ARCHITECTURE

## Authority Boundary

ALIVE Core is intentionally power-free.

The decision pipeline ends at **description**.

There are no execution paths in the core.
There are no arrows from selection → execution.

Execution is outside the current system.

```
Intent
  ↓
Eligibility (Phase 26)
  ↓
Selection Description (Phase 27)
  ↓
[ HARD STOP — NO EXECUTION ]
```
